Recent Projects

Equal Split (App Store)

Tools: CoreData, UserDefaults, UIKit

GitHub: Equal Split

GitHub: Express.js Backend


The goal was to fully develop a simple yet useful iOS application and successfuly push to App Store. This app allows to keep track of spending within a group. The user inputs the people within the group and their monetary contribution. In return, the app runs an algorithm that finds the minimum number of transactions between the people in the group necessary for the money to be split equally. ​
​
The UI was built using UIKit, both programatically and using Storyboards. CoreData is used for data persistent. The app provides fully functional logic with user-friendly animated interface and dark mode support. It is available on the App Store and is used to casually keep track of spendings offline.​

Instagram Clone

Tools: Firebase, Swift Package Manager, UIKit

GitHub: Instagram Clone


The goal was to learn and utilize Cloud Firestore REST API as well as Swift Package Manager because earlier I only had experience with CocoaPods. This Instagram clone contains Login and Registration pages allowing the user to authenticate through Firebase Authentication. Authenticated users can upload photos from their gallery using YPImagePicker dependency as well as view other users' photos. All data is stored using Firestore. ​
​
The UI was built fully programmitcally with UIKit. The project is built using MVVM architectural pattern.

SmartCaption

Tools: URLSession, UIKit, Swift

GitHub: SmartCaption


I wanted to create an application only using URLSession for REST API requests. SmartCaption allows the user a smart way to automatically caption their images through uploading the photo to a machine learning tagging endpoint from Imagga and then retrieving a fitting quote based on these tags from another open RESTful API provided by lukePeavey. ​
​
The main obstacle was sending POST requests with multipart/form-data as URLSession does not support this functionality by itself. I built this project using MVC architectural pattern with a few additional View Models for a more responsive UI.
